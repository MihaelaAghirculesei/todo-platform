# Todo Platform – Architecture & Team Work Plan

## 1. Project Goal
The goal of this project is to build a professional Todo Platform with a clear separation between backend and frontend.
The focus is on clean structure, teamwork, and company-level workflows, not on speed.

This project is intended as a learning and practice project simulating real-world software development.
